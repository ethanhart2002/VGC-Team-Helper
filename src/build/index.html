About

Version: 1.1.3


VGC Team Helper is a tool created by Ethan Hart that aims to provide exceptional analysis of a user's Pokémon team. This is done through inspecting teams for elements that have been staples in VGC throughout the format's history, such as type cores, coverage moves, modes, and supporting moves.


This tool currently only provides analysis in the context of Pokémon's official competitive format, Pokémon VGC. To use this tool, go to Pokémon Showdown, create a Pokémon team using the Teambuilder, and export your team to Pokepaste. Once you've received your Pokepaste link, copy and paste the link into this site's form to receive your analysis and grade.
